# 秩序册生成器 - 年级运动项目展示功能实现计划

## 项目分析

当前项目已经实现了从Excel文件解析学生信息和参赛项目，并生成秩序册的功能。现在需要添加一个新功能：在初步解析Excel后，将1-3年级和4-6年级的运动项目分别展示给用户。

## 实现计划

### [x] 任务1: 扩展SportsMeetManager类，添加提取年级运动项目的功能
- **优先级**: P0
- **依赖**: 无
- **描述**:
  - 在SportsMeetManager类中添加一个方法，用于提取和分类不同年级的运动项目
  - 实现1-3年级和4-6年级运动项目的分类逻辑
  - 确保方法能够返回分类后的运动项目列表
- **成功标准**:
  - 能够正确提取每个年级的运动项目
  - 能够将运动项目正确分类为1-3年级和4-6年级
  - 方法返回的数据结构清晰易读
- **测试要求**:
  - `programmatic` TR-1.1: 能够正确提取并分类运动项目
  - `programmatic` TR-1.2: 处理不同格式的Excel文件时能正常工作
- **注意事项**:
  - 考虑到可能存在的年级缺失情况
  - 确保分类逻辑清晰明确

## 技术实现要点

1. **扩展SportsMeetManager类**:
   - 添加`extract_grade_events`方法，返回分类后的运动项目
   - 实现年级分组逻辑：1-3年级为一组，4-6年级为一组
   - 确保方法能够处理不同格式的Excel文件

2. **扩展GUI界面**:
   - 在主窗口中添加新的选项卡或框架
   - 添加"预览运动项目"按钮
   - 实现运动项目的展示界面，包括年级分组和项目列表
   - 使用适当的控件（如Treeview或Listbox）展示项目

3. **数据流程**:
   - 解析Excel文件后提取运动项目
   - 分类运动项目为1-3年级和4-6年级
   - 在GUI中展示分类后的项目
   - 允许用户预览后直接生成秩序册

4. **用户体验**:
   - 提供清晰的操作流程
   - 确保界面美观、响应及时
   - 提供适当的错误提示

## 预期成果

完成后，用户将能够：
1. 选择Excel文件后，点击"预览运动项目"按钮
2. 查看1-3年级和4-6年级分别有哪些运动项目
3. 确认项目信息后，点击"生成秩序册"按钮生成最终文档
4. 获得清晰、直观的运动项目预览体验

这个功能将帮助用户在生成秩序册前了解各个年级的运动项目分布，提高用户体验和工作效率。
